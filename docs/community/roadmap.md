# 社区共建图

目标：把这个仓库做成面向全球初学者、创作者、开发者与团队的 Codex 实践知识库。

## 第一阶段：骨架完整

- [x] 建立 VuePress + Theme Hope 文档站。
- [x] 建立 README、贡献指南、行为准则和安全政策。
- [x] 搭建入门、进阶、团队和案例目录。
- [x] 添加官方资料索引。
- [ ] 补充第一批截图和命令输出。
- [ ] 给每篇教程增加“适用版本”和“最后核对日期”。

## 第二阶段：指南成型

- [ ] 完成 CLI 从安装到日常使用的完整教程。
- [ ] 完成 `AGENTS.md` 专题。
- [ ] 完成提示词模板库。
- [ ] 完成排障手册第一版。
- [ ] 补齐 Cloud、IDE、桌面 App 的使用差异。

## 第三阶段：案例驱动

- [ ] 收集 20 个真实工程案例。
- [ ] 每个案例包含任务说明、关键过程、验证方式和复盘。
- [ ] 建立案例标签：前端、后端、测试、重构、文档、CI、安全。
- [ ] 补充失败案例，说明 Codex 不适合直接处理的场景。

## 第四阶段：社区传播

- [ ] 设计更完整的首页和 Open Graph 图。
- [ ] 建立中英文关键词索引。
- [ ] 发布 GitHub Pages。
- [ ] 整理适合社媒传播的短链接和学习路线图。
- [ ] 邀请早期贡献者认领章节。

## 贡献入口

你可以从这些任务开始：

- 修正文档里的不准确表述。
- 新增一个你亲自跑过的 Codex 案例。
- 给排障手册补一个常见错误。
- 把官方文档的新变化同步到资料索引。
